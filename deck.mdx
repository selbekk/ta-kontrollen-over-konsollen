## Selvdokumenterende kode ftw

```js
console.debug('Kun for spesielt interesserte');
console.info('Fortell hva som skjer i koden din!');
console.warn('Her er ikke ting helt på stell');
console.error('Shit pomfritt! 💩🍟');
```

```notes
Da vi var ferske fikk vi vite at koden vår burde være selvdokumenterende!
La oss gjøre som i java, og dokumentere med kommentara!

Vi har loggnivåer innebygget i språket!
- warn og error kommer med stack trace
```

## Woop woop console.group

```js
console.group('action');
console.info('previous state', state);
console.info('action payload', action);
console.info('next state', reducer(state, action));
console.groupEnd('action');
```

```notes
Har dere brukt Redux? Redux Logger?
Kan komme mye informasjon, smart å gruppere!
console.group lar deg gruppere flere lag innover!
```

## It's all about the $$

```js
$('#css-selector')
$$('.css-selector')
```

```notes
jQuery er kjempegammelt, men det var diggere å kunne skrive $ enn document.querySelector.
I consolen kan du fortsatt leve som det var 2009
Funksjonen $ er et alias for document.querySelector 
Funksjonen $$ er et alias for document.querySelectorAll
```

## Inspektør! 🕵️‍

```js

```

```notes
Har du funnet et DOM-objekt du vil inspisere nærmere, kan du kalle inspect-funksjonen.
Da åpnes DOM-inspektøren, og fokuserer elementet du oppga.

Perfekt for når du debugger CSS og er for lat til å bruke musepekeren.
```
